# dotfiles
> That's the Way (I Like It)

## Requirements

- zsh
- [prezto](https://github.com/sorin-ionescu/prezto)
- [ripgrep](https://github.com/BurntSushi/ripgrep)
- [fasd](https://github.com/clvv/fasd)

## Install

```sh
git clone https://github.com/marksteve/dotfiles
cd dotfiles
./install.sh
```
